#CSS GRID

##Conceitos estudados

O que é o CSS Grid: sistema de layout bidimensional para organizar elementos em colunas e linhas.

Gaps: espaçamentos entre as colunas e as linhas do grid usando gap, row-gap e column-gap.

Colunas e linhas: definição da estrutura do grid com grid-template-columns e grid-template-rows.

Unidade fr
fr: unidade fracionária que divide o espaço disponível entre as colunas/linhas.

repeat(): forma de repetir padrões de colunas/linhas sem precisar escrever tudo manualmente.

minmax(): define um tamanho mínimo e máximo para colunas/linhas, ajudando na responsividade.

Posicionamento no Grid: posicionar itens usando grid-column, grid-row e/ou grid-area.

Mesclando itens em linhas/colunas: fazer um item ocupar mais de uma coluna/linha usando span.

grid-area: atalho para definir início/fim de linha e coluna (ou nomear uma área no template).

grid-template-areas: cria um “mapa” visual de áreas do layout usando nomes.

Estrutura de layout em Grid: montar páginas com regiões (header, menu, conteúdo, footer) usando áreas e templates.

Responsividade no Grid: adaptar o grid para diferentes telas mudando colunas, áreas e tamanhos com media queries e funções como minmax().
